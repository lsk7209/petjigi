## 2. 어필리에이트 표시 의무 (표시광고법·공정거래위원회 추천보증 심사지침)

### 2.1 의무 표시 문구 (어필리에이트 별)

**쿠팡파트너스:**
> "이 페이지는 쿠팡파트너스 활동의 일환으로, 이에 따른 일정액의 수수료를 제공받습니다."

**비마이펫·라이펫·기타 펫보험 어필리에이트:**
> "이 페이지에는 제휴 링크가 포함되어 있으며, 사용자가 링크를 통해 가입 또는 구매 시 펫지기가 일정 수수료를 받을 수 있습니다. 콘텐츠의 객관성에는 영향을 주지 않습니다."

**일반 어필리에이트:**
> "이 페이지에는 제휴 링크가 포함되어 있습니다. 자세한 내용은 [어필리에이트 고지](/disclosure) 페이지를 참고해 주세요."

### 2.2 표시 위치 의무

| 위치 | 의무 |
|------|------|
| **페이지 상단** (스크롤 없이 보이는 영역) | ✓ 필수 |
| 어필리에이트 링크 직전 또는 직후 | ✓ 필수 (작은 라벨 "제휴 링크") |
| 페이지 푸터 | ✓ 필수 (전체 표시 문구) |
| /disclosure 별도 페이지 | ✓ 필수 (전체 정책) |

### 2.3 링크 기술 속성

```html
<a href="..." rel="sponsored nofollow" 
   data-affiliate="coupang-partners"
   class="affiliate-link">
  ...
  <span class="affiliate-label">제휴 링크</span>
</a>
```

- 모든 어필리에이트 링크: `rel="sponsored nofollow"`
- `data-affiliate` 속성으로 추적 분류
- 시각적 라벨 "제휴 링크" 또는 아이콘 (※)

## 3. 개인정보 동의 흐름 (개인정보보호법)

### 3.1 이메일 수집 폼 표준

```
┌─────────────────────────────────────────────────────┐
│ 무료 가이드 PDF 받기                                 │
│                                                     │
│ 이메일 주소: [____________________]                  │
│                                                     │
│ [필수] ☐ 개인정보 수집·이용에 동의합니다 [내용 보기]  │
│                                                     │
│ [선택] ☐ 광고성 정보 수신에 동의합니다 [내용 보기]    │
│                                                     │
│ [PDF 받기] (필수 동의 시 활성화)                     │
└─────────────────────────────────────────────────────┘
```

**중요**: 두 동의는 반드시 **분리 체크박스**. 묶음 동의 금지.

### 3.2 동의 문구 표준

**필수 동의:**
> "**개인정보 수집·이용 동의 (필수)**
> 
> - 수집 항목: 이메일 주소
> - 수집 목적: 무료 가이드 PDF 발송, 서비스 안내
> - 보관 기간: 회원 탈퇴 또는 발송 동의 철회 시까지
> - 동의 거부 권리: 동의를 거부할 수 있으며, 거부 시 가이드 발송이 불가합니다.
> 
> 자세한 사항은 [개인정보처리방침](/privacy)을 확인해 주세요."

**선택 동의:**
> "**광고성 정보 수신 동의 (선택)**
> 
> 펫지기의 신규 콘텐츠·가이드·이벤트 안내 등 광고성 정보 수신에 동의합니다.
> 
> - 수신 거부: 모든 이메일 하단 'unsubscribe' 링크로 즉시 철회 가능
> - 선택 사항: 미동의 시에도 가이드 PDF는 정상 발송됩니다."

### 3.3 14세 미만 보호 (만일 대비)

- 이메일 수집 폼에 "**본인은 만 14세 이상입니다**" 추가 체크박스 (필수)
- 미체크 시 가입 불가
- 14세 미만이 입력해도 우리는 책임 회피 가능 (법적 보호 장치)

### 3.4 이메일 발송 시 필수 사항

모든 발송 이메일 푸터에:
- 발신자 정보 (사이트명·주소)
- 수신 거부 (unsubscribe) 링크 (1-click)
- 개인정보 처리방침 링크
- 광고성 정보인 경우 "(광고)" 제목 머리말

## 4. 필수 법적 문서 골격

### 4.1 필요한 페이지 4종

| URL | 내용 |
|-----|------|
| `/terms` | 이용약관 |
| `/privacy` | 개인정보처리방침 |
| `/disclosure` | 어필리에이트 고지 |
| `/advertising` | 광고 게재 정책 |

### 4.2 개인정보처리방침 필수 항목

1. 처리하는 개인정보 항목
   - 이메일 주소 (수집)
   - IP 주소, 쿠키, User-Agent (자동 수집)
2. 처리 목적
   - 가이드 발송
   - 서비스 운영 및 개선
   - 광고성 정보 발송 (별도 동의 시)
3. 처리 및 보유 기간
4. 처리 위탁 (제3자 처리 위탁)
   - Resend (이메일 발송, 미국)
   - Google Analytics (분석, 미국)
   - Google AdSense (광고, 미국)
   - Vercel (호스팅, 미국)
   - Turso (DB, 미국)
5. 개인정보의 국외 이전
   - 위 모두 미국 등 해외 클라우드 사용
   - 이전 국가, 이전 일시, 항목, 이용 목적, 보유 기간 명시
6. 정보주체의 권리
   - 열람, 정정, 삭제, 처리 정지 요구
7. 자동 수집 장치 운영·거부 방법
   - 쿠키 설정 안내
8. 개인정보 보호책임자
   - 이름, 직책, 연락처 (이메일)
9. 변경 고지
   - 변경 시 7일 전 사이트 공지

### 4.3 어필리에이트 고지 (/disclosure) 골격

```markdown
# 어필리에이트 고지

펫지기는 일부 페이지에 어필리에이트(제휴) 링크를 포함합니다. 본 페이지에서 펫지기가 어떻게 수익을 얻는지, 그것이 콘텐츠에 어떤 영향을 미치는지 투명하게 안내합니다.

## 어필리에이트 활동 사실
펫지기는 다음 어필리에이트 프로그램에 참여하고 있습니다:
- 쿠팡파트너스 (펫푸드·메모리얼 굿즈 등)
- 비마이펫 어필리에이트 (펫보험 비교)
- 라이펫 어필리에이트 (펫보험)
- (Phase 3+) 펫서비스 예약 플랫폼

## 수수료 발생 조건
사용자가 펫지기의 어필리에이트 링크를 통해 해당 사이트로 이동한 후 구매·가입·예약 등의 행동을 완료할 경우, 펫지기는 일정 비율의 수수료를 받습니다. 수수료는 사용자가 추가로 부담하는 비용이 아닙니다.

## 추천의 객관성
어필리에이트 수수료가 콘텐츠의 객관성에 영향을 주지 않도록 다음 원칙을 따릅니다:
- 추천 기준은 어필리에이트 가입 여부가 아닌 사용자 가치 기준입니다
- 비교 콘텐츠는 어필리에이트 가입 여부와 무관하게 모든 주요 옵션을 다룹니다
- 부정적인 정보도 동일하게 다룹니다

## 어필리에이트 링크의 표시
어필리에이트 링크는 다음과 같이 명확히 표시됩니다:
- 페이지 상단의 어필리에이트 고지
- 링크 근처의 "제휴 링크" 라벨
- HTML rel="sponsored nofollow" 속성

## 문의
어필리에이트 관련 문의는 contact@petjigi.com 으로 연락 주세요.

최종 업데이트: 2026-MM-DD
```
